# Projeto: Casa das Latinas
Descrição: Site exemplo (HTML5, CSS3, JS) para plataforma de apoio a mulheres estrangeiras latinas no Brasil.
Estrutura:
  - index.html
  - projetos.html
  - cadastro.html
  - css/styles.css
  - js/scripts.js
  - images/*.svg

Instruções:
  1. Descompacte o arquivo.
  2. Faça upload da pasta para um repositório público no GitHub.
  3. Ative GitHub Pages (opcional) apontando para a branch principal ou pasta / (root).
  4. Para uso em produção, conecte a um backend para processar envios do formulário com validações servidoras.

Acessibilidade e técnicas aplicadas:
  - HTML5 semântico
  - Formulários com validação nativa e máscaras via JS
  - Design responsivo mobile-first
  - Paleta em tons de vermelho e variações
  - Animações suaves via CSS
